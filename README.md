# 🎙️ مشروع بث القرآن الكريم 24/7 على تيليجرام

هذا المشروع مصمم ليعمل بكفاءة واستقرار بنسبة 100% على منصة **Railway** عبر ربطه بمستودع **GitHub**. تم تعديل الكود ليعتمد بالكامل على **متغيرات البيئة (Environment Variables)** لحماية بياناتك السرية ومنع حظر حسابك عند رفعه على GitHub.

## 📂 محتويات المشروع:
1. `app.py`: الكود البرمجي الأساسي للبث الصوتي باستخدام `Pyrogram` و `pytgcalls`.
2. `Dockerfile`: ملف إعداد البيئة وتثبيت أداة `ffmpeg` لمعالجة الصوت بدون أخطاء.
3. `requirements.txt`: المكتبات البرمجية المطلوبة بالإصدارات المتوافقة.

## ⚙️ المتغيرات المطلوبة في منصة Railway:
عند رفع المشروع على Railway، يجب عليك إضافة المتغيرات التالية في قسم **Variables**:

* `API_ID`: رقم الـ API الخاص بحسابك (تحصل عليه من my.telegram.org).
* `API_HASH`: كود الـ API Hash الخاص بحسابك (تحصل عليه من my.telegram.org).
* `SESSION_STRING`: كود جلسة الحساب (String Session) المستخرج لحساب المساعد.
* `CHAT_ID`: آيدي القناة أو المجموعة التي سيتم البث فيها (يجب أن يبدأ بـ `100-` للأقنية والمجموعات الخارقة).
* `RADIO_URL`: (اختياري) رابط بث الإذاعة، تم وضع إذاعة الترتيل كخيار افتراضي.

## 🚀 طريقة التشغيل:
1. ارفع هذه الملفات إلى مستودع جديد على GitHub.
2. اربط المستودع بمنصة Railway وسيتم التشغيل تلقائياً.
